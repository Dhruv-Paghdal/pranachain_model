To test and run the project on your system follow the below steps:

1. Run the data cleaning code to generate the CSV files.
2. Run the CKD\_Model.ipynb file to train and save the model.
3. Test the saved model performance using the CKD\_Model\_Testing.ipynb file.
4. Run "streamlit run app.py" and access the model through Streamlit UI. Upload patient\_report\_1.txt or patient\_report\_1.txt file to see the output.
